# SkillTree Vis - 开发者技能树可视化系统

一个完整的开发者技能树可视化项目，包含数据采集、分析和可视化展示。

## 📁 项目结构

```
SkillTree Vis/
├── index.html                  # 前端可视化页面（增强版）
├── graph_data.json             # 综合技能树数据
├── data_python_fullstack.json  # Python 全栈开发数据集
├── data_java_backend.json      # Java 后端开发数据集
├── data_data_analyst.json      # 数据分析师数据集
├── datasets.json               # 多数据集配置
├── simple_analyzer.py          # 简化版 Python 分析器（无依赖）
├── generate_all_datasets.py    # 批量生成数据集脚本
├── skill_bank.json             # 技能词库
├── sample_jds.json             # 示例 JD 数据
├── requirements.txt            # Python 依赖
└── README.md                   # 项目文档
```

## 🚀 快速开始

### 1. 查看可视化效果

启动本地服务器（Python 3.x）：

```bash
python -m http.server 8000
```

然后访问 http://localhost:8000 即可！

### 2. 运行 Python 分析脚本（无需依赖）

直接运行简化版分析器（无需安装任何依赖）：

```bash
python simple_analyzer.py
```

这将读取 `sample_jds.json` 中的示例数据，分析后生成新的 `graph_data.json` 文件。

### 3. 批量生成所有数据集

运行以下命令生成所有岗位的数据集：

```bash
python generate_all_datasets.py
```

这将生成：
- `data_python_fullstack.json` - Python 全栈开发
- `data_java_backend.json` - Java 后端开发
- `data_data_analyst.json` - 数据分析师

## 💡 使用说明

### 基础操作

1. **切换数据集**：在顶部下拉菜单选择不同岗位的技能树
2. **搜索技能**：在顶部搜索框输入技能名称，按回车快速定位
3. **查看排行榜**：点击侧边栏的排行榜项快速定位技能
4. **分类筛选**：选择分类并点击"应用筛选"查看特定类别的技能
5. **查看关联**：点击任意技能节点，高亮显示关联技能
6. **切换视图**：点击"力导向图"或"卡片视图"切换显示模式
7. **调整视图**：拖拽节点调整位置，滚轮缩放视图
8. **切换主题**：点击"🌙 暗色"/"☀️ 亮色"切换深色/亮色主题
9. **重置**：点击"重置"按钮恢复初始状态

### 学习追踪功能

#### 学习时间追踪
1. 点击任意技能节点
2. 在侧边栏的技能详情中点击"▶️ 开始学习"
3. 计时器开始计时
4. 点击"⏹️ 结束学习"停止计时并保存学习记录
5. 查看侧边栏的"学习时间"统计

#### 学习进度管理
1. 点击任意技能节点
2. 在侧边栏选择学习进度（📚 待学习 / 🎯 学习中 / ✅ 已完成）
3. 进度会自动保存到本地存储
4. 侧边栏的"学习进度"会实时更新

#### 个人笔记
1. 点击任意技能节点
2. 在侧边栏的"📝 个人笔记"区域输入笔记
3. 笔记会自动保存
4. 卡片视图中会显示笔记指示器

#### 技能权重
1. 点击任意技能节点
2. 拖动"⚖️ 技能权重"滑块调整节点大小
3. 权重范围：0.5x - 2x
4. 力导向图会实时更新

### 学习资源管理

#### 添加学习资源
1. 点击任意技能节点
2. 在侧边栏点击"➕ 添加学习资源"
3. 填写资源信息：
   - 资源名称（必填）
   - 资源链接（可选）
   - 资源类型（教程/文档/视频/书籍/其他）
4. 点击"保存"按钮

#### 管理资源
- 点击"🔗 打开"在新标签页打开资源链接
- 点击"🗑️"删除该资源
- 查看推荐资源列表

### 技能熟练度评分

#### 添加/修改评分
1. 点击任意技能节点
2. 在侧边栏点击"➕ 添加/修改评分"
3. 点击星星选择评分（1-5星）
4. 可选：添加评价备注
5. 点击"保存评分"

#### 查看评分历史
- 侧边栏显示当前评分星级
- 查看评分历史记录（最多显示5条）
- 评分历史包含时间、星级和备注

### 导出功能

#### 使用增强导出
1. 点击顶部导航栏的"📦 导出"按钮
2. 选择导出格式：
   - 📊 CSV (Excel) - 适合用表格软件查看
   - 🌐 HTML 报告 - 美观的学习报告
   - 📋 JSON 数据 - 结构化数据
3. 选择需要导出的内容（可选）：
   - 技能信息
   - 学习进度
   - 学习时间
   - 个人笔记
   - 学习资源
   - 技能熟练度
4. 点击"开始导出"

#### 其他导出选项
- 📷 导出图片：导出力导向图为 PNG 图片
- 📄 导出报告：导出 Markdown 格式的学习报告
- 💾 导出 JSON：导出原始 JSON 数据

### 分享与导入

#### 分享学习状态
1. 点击"🔗 分享状态"按钮
2. 学习状态编码会自动复制到剪贴板
3. 将编码发送给好友

#### 导入学习状态
1. 在侧边栏的"📥 导入学习状态"区域
2. 点击或粘贴分享的编码
3. 学习进度、笔记、权重会自动恢复

#### 导入自定义数据
1. 在侧边栏的"📁 导入数据"区域
2. 点击或拖拽 JSON 文件到导入区域
3. 数据格式需与 graph_data.json 一致

### 技能对比

1. 点击任意技能节点
2. 点击"➕ 添加到对比"按钮
3. 重复添加最多 5 个技能
4. 在侧边栏查看对比表格
5. 点击"×"从对比列表中移除技能
6. 点击"清空对比"清空整个列表

### 智能学习路径

1. 点击任意技能节点查看相关技能
2. 未选择节点时查看推荐学习顺序
3. 推荐顺序基于：
   - 学习进度（优先推荐未学习技能）
   - 技能难度（从简单到复杂）
   - 技能重要性（出现次数）
   - 熟练度评分

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `/` 或 `Ctrl+K` | 聚焦搜索框 |
| `G` | 切换到力导向图视图 |
| `C` | 切换到卡片视图 |
| `R` | 重置视图 |
| `T` | 切换主题（暗色/亮色） |
| `1-4` | 切换数据集 |
| `Esc` | 取消选择/关闭高亮 |

### 自定义数据

1. 编辑 `datasets.json` 添加你自己的岗位数据集
2. 编辑 `skill_bank.json` 扩展技能词库
3. 运行 `python generate_all_datasets.py` 重新生成所有图数据
4. 在前端下拉菜单中选择新数据集查看效果

## 🛠️ 技术栈

- **前端**：HTML5 + CSS3 + ECharts.js 5.4.3
- **后端**：Python 3.8+
- **数据分析**：仅使用 Python 标准库（json, collections）
- **数据存储**：localStorage（浏览器本地存储）
- **数据格式**：JSON

## 📊 分析算法说明

1. **关键词提取**：基于技能词库进行精确匹配
2. **共现矩阵**：统计技能在同一 JD 中的共同出现次数
3. **简化 TF-IDF**：计算技能的重要性权重（无外部依赖）
4. **节点大小**：综合出现次数和 TF-IDF 分数计算
